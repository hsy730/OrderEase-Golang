# OrderEase API 文档索引

## 文档说明

本目录包含 OrderEase 系统的 API 文档，按照功能模块拆分为以下独立文件，方便查看和维护：

## 功能模块文档列表

| 功能模块 | 文档文件 | 描述 |
|---------|---------|------|
| 商品管理 | [api_product.md](./api_product.md) | 产品创建、更新、查询等相关接口 |
| 订单管理 | [api_order.md](./api_order.md) | 订单创建、查询、删除等相关接口 |
| 用户管理 | [api_user.md](./api_user.md) | 用户创建、删除、查询等相关接口 |
| 标签管理 | [api_tag.md](./api_tag.md) | 商品标签查询等相关接口 |
| 店铺管理 | [api_shop.md](./api_shop.md) | 店铺创建、更新、查询等相关接口 |

## 文档规范

每个 API 文档文件包含以下内容：
- 接口名称和描述
- HTTP 请求方法和路径
- 请求参数说明
- 响应格式和示例
- 错误处理说明

## 版本说明

- 当前文档版本：1.0
- 最后更新时间：2024-01-01

## 注意事项

- 所有 API 请求均需按照文档指定的格式进行
- 接口返回数据格式统一为 JSON
- 接口调用需要根据权限要求进行身份认证
