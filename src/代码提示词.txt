# 全局环境配置
当前环境：Windows PowerShell 5 + Go
命令格式：在PowerShell环境下，命令连接符使用 `;` 而不是 `&&`
依赖管理命令：`cd src; go mod tidy`
启动应用命令：`cd src; go run main.go`

# 项目要求
现在的环境是go， 要求使用gorm框架，编写外卖商城服务端的接口。

# 数据库配置
我的数据库连接方式是 mysql
